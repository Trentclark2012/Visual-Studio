Plugins

  • Visual Assist: helps you with multiple aspects of coding, things like code correcting, debugging assist and Code generation.
  • Snippet Designer: Allows for a richer and more productive code snippet experience. Allows the user to be able to highlight code he wants snipped and “Exports as Snippet” bringing it directly to the snippet editor
  • ReSharper: Automates most of what can be automated in your coding routines. Compiler errors, runtime errors, redundancies right as you type. Also nice cleanup feature that gets rid of unused code to stay up to coding standards.
